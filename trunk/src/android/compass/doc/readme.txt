Compass

Die Anwendung stellt einen einfachen Kompass dar. Sie erfordert
keinerlei Rechte auf dem Gerät, blendet keine Werbung ein und
sammelt keine Daten des Nutzers. Gleichzeitig ist sie so programmiert
worden, dass sie bei Nichtbenutzung keine Ressourcen beansprucht.
Der Sensor des Kompasses wird dann sofort freigegeben und erst beim
Aktivieren der Anwendung wieder eingeschaltet.
